## Interaction States

### Table Card States

**Default**
- Light background (card style from theme)
- All buttons visible
- Description visible if present

**Expanded**
- Same as default
- Plus: Entries list visible below
- Expand button shows â–² instead of â–¼

**Hovering Roll Button**
- Cursor changes to pointer
- Button may darken slightly (theme-dependent)

**Rolling Animation**
- Roll button disabled for 300ms
- Brief visual feedback
- Result appears in sidebar

**Disabled Roll**
- Button grayed out
- Occurs when: already rolling OR table has 0 entries

### Form States

**Valid**
- Submit button enabled
- Normal styling

**Invalid**
- Submit button disabled (grayed)
- Occurs when: no title OR no entries with text

**Entry Removal**
- Remove button disabled if only 1 entry
- Grayed out with not-allowed cursor

## Animation Timing

- **Roll animation**: 300ms delay before showing result
- **Modal open/close**: Instant (no animation, for performance)
- **Expand/collapse**: Instant content toggle
- **Hover effects**: CSS-driven, instant

## Accessibility Features

- **Keyboard navigation**: Tab through all interactive elements
- **Focus indicators**: Visible borders on focused elements  
- **Button labels**: Clear text (not just icons)
- **Alt text**: Icons have title attributes for tooltips
- **Color contrast**: Meets WCAG AA in both themes

## Dark Mode Differences

**Light Mode**
- White/light gray backgrounds
- Dark text
- Subtle shadows

**Dark Mode**
- Dark gray/black backgrounds
- White/light text
- Borders for separation (less shadow reliance)
- Primary color unchanged for consistency
